GROMACSMD生体分子

1. GROMACS とは

GROMACS（GROningen MAchine for Chemical Simulations）は， 主に生体分子（タンパク質・脂質・核酸）のシミュレーションに特化した高性能 MD ソフトウェアです． AMBER・CHARMM・GROMOS・OPLS などの主要な力場に対応しています．

特徴内容
ライセンスLGPL（オープンソース）
得意な系水溶液中のタンパク質・膜・ハイドレート
並列化MPI + OpenMP + GPU (CUDA/OpenCL)
解析豊富な組み込み解析ツール（gmx コマンド群）

2. インストール

Ubuntu (WSL2)
# conda からインストール（推奨）
conda install -c conda-forge gromacs

# または apt から
sudo apt install gromacs

# バージョン確認
gmx --version

3. ファイル構成

拡張子ファイル名内容
.grostructure.gro原子座標（GROMOS 形式），ボックスサイズ
.toptopol.topトポロジー（質量・電荷・結合・力場パラメータ）
.itpmolecule.itp分子ごとのトポロジー（.top から include）
.mdpmd.mdpMD パラメータ（時間刻み・アンサンブル・出力間隔等）
.tprtopol.tprバイナリ実行ファイル（grompp が生成）
.trrtraj.trr完全なトラジェクトリ（座標・速度・力）
.xtctraj.xtc圧縮トラジェクトリ（座標のみ）
.edrener.edrエネルギー・温度・圧力の時系列
.logmd.log実行ログ

4. 基本ワークフロー

一般的な GROMACS の計算手順：

1
初期構造の準備（PDB から .gro に変換）
Terminal
gmx pdb2gmx -f protein.pdb -o protein.gro -water spce -ff amber99sb
2
ボックスと溶媒の設定
Terminal
# 立方体ボックスを作成（分子から 1.0 nm のマージン）
gmx editconf -f protein.gro -o box.gro -c -d 1.0 -bt cubic

# ボックスに水分子を充填
gmx solvate -cp box.gro -cs spc216.gro -o solvated.gro -p topol.top

# イオンの追加（電荷中性化）
gmx grompp -f ions.mdp -c solvated.gro -p topol.top -o ions.tpr
gmx genion -s ions.tpr -o ionized.gro -p topol.top -pname NA -nname CL -neutral
3
エネルギー最小化
Terminal
gmx grompp -f em.mdp -c ionized.gro -p topol.top -o em.tpr
gmx mdrun -v -deffnm em
4
NVT 平衡化（温度を目標値に緩和）
Terminal
gmx grompp -f nvt.mdp -c em.gro -r em.gro -p topol.top -o nvt.tpr
gmx mdrun -deffnm nvt
5
NPT 平衡化（圧力を目標値に緩和）
Terminal
gmx grompp -f npt.mdp -c nvt.gro -r nvt.gro -t nvt.cpt -p topol.top -o npt.tpr
gmx mdrun -deffnm npt
6
本計算（Production MD）
Terminal
gmx grompp -f md.mdp -c npt.gro -t npt.cpt -p topol.top -o md.tpr
gmx mdrun -deffnm md

5. mdp ファイルの設定

md.mdp — 本計算用パラメータ（NPT, 300K, 1bar）
; 実行制御
integrator  = md          ; リープフロッグ積分
nsteps      = 50000000    ; ステップ数（× 2 fs = 100 ns）
dt          = 0.002       ; 時間刻み [ps]

; 出力制御
nstxout-compressed = 5000   ; XTC 出力間隔（10 ps ごと）
nstenergy          = 5000   ; エネルギー出力間隔
nstlog             = 5000   ; ログ出力間隔

; 結合拘束（LINCS）
constraints         = h-bonds  ; H 含む結合を拘束
constraint-algorithm = lincs

; 近距離相互作用
cutoff-scheme = Verlet
nstlist       = 10
ns-type       = grid
rlist         = 1.0

; 静電気（PME）
coulombtype   = PME
rcoulomb      = 1.0
ewald-rtol    = 1e-5

; van der Waals
vdwtype       = Cut-off
rvdw          = 1.0

; 温度制御（V-rescale, 300 K）
tcoupl          = V-rescale
tc-grps         = Protein Non-Protein
tau-t           = 0.1 0.1
ref-t           = 300 300

; 圧力制御（Parrinello-Rahman, 1 bar）
pcoupl          = Parrinello-Rahman
pcoupltype      = isotropic
tau-p           = 2.0
ref-p           = 1.0
compressibility = 4.5e-5

; PBC
pbc             = xyz

6. 解析コマンド

よく使う解析コマンド
# エネルギー・温度・圧力の抽出
gmx energy -f md.edr -o energy.xvg

# RMSD（タンパク質の構造変化）
gmx rms -s md.tpr -f md.xtc -o rmsd.xvg

# RMSF（残基ごとの揺らぎ）
gmx rmsf -s md.tpr -f md.xtc -o rmsf.xvg -res

# 動径分布関数（RDF）
gmx rdf -f md.xtc -s md.tpr -o rdf.xvg -ref "resname SOL and name OW" -sel "resname SOL and name OW"

# MSD（拡散係数）
gmx msd -f md.xtc -s md.tpr -o msd.xvg -lateral z

# 水素結合数
gmx hbond -f md.xtc -s md.tpr -num hbnum.xvg

# 回転自己相関関数
gmx rotacf -f md.xtc -s md.tpr -o rotacf.xvg

# トラジェクトリの可視化（VMD, OVITO 用に変換）
gmx trjconv -f md.xtc -s md.tpr -o md_pbc.xtc -pbc mol -center
