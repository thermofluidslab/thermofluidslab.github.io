1. AIMD とは

第一原理分子動力学法（Ab initio Molecular Dynamics, AIMD）は，古典的な力場を使わず， 密度汎関数理論（DFT）などの第一原理電子状態計算によってその場で原子間力を求めながら ニュートンの運動方程式を解く MD 手法です．

古典 MD に比べて計算コストは数桁大きいですが，化学結合の形成・切断， 電子的効果，分極，vdW 相互作用（補正付き）などを直接取り扱えます． 当研究室では主にクラスレートハイドレートの振動スペクトル・ゲスト-ホスト相互作用の解析に用いています．

手法力の計算適用規模代表ソフト
古典 MD力場（パラメトリック）〜10⁶ 原子LAMMPS, GROMACS
ML-FF MD機械学習ポテンシャル〜10⁵ 原子LAMMPS + MACE
AIMD (BO-MD)DFT（毎ステップ）〜100 原子VASP, CP2K
AIMD (CP-MD)DFT（拡張ラグランジアン）〜100 原子CPMD, CP2K

2. DFT の基礎

密度汎関数理論（DFT）は，多電子系のシュレーディンガー方程式を 電子密度 $\rho(\mathbf{r})$ の汎関数として解く方法です．Kohn–Sham 方程式：

$$\left[ -\frac{\hbar^2}{2m}\nabla^2 + V_\text{ext}(\mathbf{r}) + V_\text{Hartree}(\mathbf{r}) + V_\text{xc}(\mathbf{r}) \right] \psi_i(\mathbf{r}) = \varepsilon_i \psi_i(\mathbf{r})$$

$V_\text{xc}$ が交換相関汎関数で，近似の違いが計算精度に大きく影響します：

分類例特徴
LDAPW92過結合傾向，高速
GGAPBE, BLYP固体・分子に広く使用
GGA+DPBE-D3, revPBE-D3vdW 分散力補正あり（ハイドレートに必要）
HybridPBE0, HSE06高精度，計算コスト大
meta-GGASCAN, r²SCAN水・氷の記述に優れる
vdW 補正について： クラスレートハイドレートは水素結合に加えてロンドン分散力が重要です． GGA 汎関数（PBE 等）は分散力を過小評価するため，DFT-D3（Grimme の補正）を使うことが重要です（当研究室の論文でも PBE-D3 を使用）．

3. Born-Oppenheimer MD（BO-MD）

Born-Oppenheimer 近似のもと，各 MD ステップで核の配置を固定し， 電子系を完全に収束させてから原子に働く力 $\mathbf{F}_I = -\nabla_{\mathbf{R}_I} E_\text{DFT}$ を計算します．

$$M_I \ddot{\mathbf{R}}_I = -\nabla_{\mathbf{R}_I} \min_{\{\psi_i\}} E_\text{DFT}[\rho]$$

各ステップで SCF（自己無撞着場）計算を収束させる必要があるため， 1ステップあたりの計算コストが大きいですが，実装が単純で安定性が高いため， 現在の主流です（VASP の AIMD はこの方式）．

4. Car-Parrinello MD（CP-MD）

Car と Parrinello（1985）が提案した手法で，電子の自由度（Kohn-Sham 軌道係数）を 原子と同様に「仮想的な質量」を持つ古典変数として扱う拡張ラグランジアン法です．

$$\mathcal{L}_\text{CP} = \frac{1}{2}\sum_I M_I \dot{\mathbf{R}}_I^2 + \frac{\mu}{2}\sum_i \langle\dot\psi_i|\dot\psi_i\rangle - E_\text{DFT}[\{\psi_i\}, \{\mathbf{R}_I\}]$$

SCF 計算を各ステップで行わず，軌道を連続的に伝播させるため高速ですが， 仮想質量 $\mu$ の適切な選択や時間刻みの制約があります． 現在は BO-MD の方が広く使われていますが，CP2K は両方をサポートします．

5. VASP による AIMD

INCAR の設定例（NVT アンサンブル）

INCAR — AIMD（NVT, 300 K）
SYSTEM = clathrate hydrate AIMD

# 基本設定
ISTART = 0         # 新規計算（0）/ 継続（1）
ICHARG = 2         # 原子の重ね合わせから電荷密度初期化
ENCUT  = 400       # プレーンウェーブカットオフ [eV]
EDIFF  = 1E-5      # SCF 収束判定
PREC   = Normal

# 交換相関汎関数 (PBE + D3 vdW 補正)
GGA    = PE        # PBE 汎関数
IVDW   = 11        # DFT-D3 (Grimme 2010)

# MD 設定
IBRION = 0         # MD モード
NSW    = 20000     # MD ステップ数
POTIM  = 0.5       # 時間刻み [fs]
SMASS  = 0         # Nosé–Hoover サーモスタット
TEBEG  = 300       # 初期温度 [K]
TEEND  = 300       # 終了温度 [K]

# アンサンブル
MDALGO = 2         # Nosé–Hoover (NVT)
ISYM   = 0         # 対称性を使わない（MD では必須）

# 出力制御
NWRITE = 0         # 出力頻度最小化
NBLOCK = 1         # XDATCAR への書き出し間隔
LWAVE  = .FALSE.   # WAVECAR を書かない
LCHARG = .FALSE.   # CHGCAR を書かない

# k点（Γ点のみで十分な場合）
KSPACING = 1000    # 実質 Γ 点のみ

計算の流れ

1
構造最適化（IBRION=2, NSW=100 程度）で初期構造を緩和
2
速度初期化：VASP は TEBEG から Maxwell-Boltzmann 分布で速度を自動設定
3
平衡化（1〜2 ps 程度，速度スケーリングまたはサーモスタット使用）
4
本計算（10〜50 ps，OSZICAR・XDATCAR・OUTCAR に記録）

出力ファイル

ファイル内容
XDATCAR全ステップの原子座標（軌跡）
OSZICAR各ステップの温度・エネルギー
OUTCAR詳細な計算情報（力，ストレス等）
vasprun.xml全情報を含む XML（ASE で読み込み可）

6. CP2K による AIMD

CP2K は Gaussian と平面波（GPW）の混合基底を使うオープンソースの AIMD コードです． 大規模系でも効率よく計算でき，AIMD・NMR・Born-Effective-Charge などに対応しています．

cp2k.inp — AIMD（NVT, 300 K）の最小例
&GLOBAL
  PROJECT  hydrate_md
  RUN_TYPE MD
  PRINT_LEVEL LOW
&END GLOBAL

&MOTION
  &MD
    ENSEMBLE   NVT
    STEPS      10000
    TIMESTEP   0.5        ! [fs]
    TEMPERATURE 300
    &THERMOSTAT
      TYPE NOSE
      &NOSE
        TIMECON 100.0
      &END NOSE
    &END THERMOSTAT
  &END MD
  &PRINT
    &TRAJECTORY
      &EACH
        MD 1
      &END EACH
    &END TRAJECTORY
    &VELOCITIES ON
    &END VELOCITIES
  &END PRINT
&END MOTION

&FORCE_EVAL
  METHOD Quickstep
  &DFT
    BASIS_SET_FILE_NAME  BASIS_MOLOPT
    POTENTIAL_FILE_NAME  GTH_POTENTIALS
    &QS
      METHOD GPW
      EPS_DEFAULT 1.0E-10
    &END QS
    &MGRID
      CUTOFF 400
      REL_CUTOFF 40
    &END MGRID
    &XC
      &XC_FUNCTIONAL PBE
      &END XC_FUNCTIONAL
      &VDW_POTENTIAL          ! DFT-D3 vdW 補正
        POTENTIAL_TYPE PAIR_POTENTIAL
        &PAIR_POTENTIAL
          TYPE DFTD3
          PARAMETER_FILE_NAME dftd3.dat
          REFERENCE_FUNCTIONAL PBE
        &END PAIR_POTENTIAL
      &END VDW_POTENTIAL
    &END XC
    &SCF
      SCF_GUESS ATOMIC
      EPS_SCF 1.0E-6
      MAX_SCF 50
    &END SCF
  &END DFT
  &SUBSYS
    &CELL
      ABC 12.0 12.0 12.0    ! ボックスサイズ [Å]
    &END CELL
    &TOPOLOGY
      COORD_FILE_NAME  init.xyz
      COORD_FILE_FORMAT XYZ
    &END TOPOLOGY
    &KIND O
      BASIS_SET  DZVP-MOLOPT-GTH
      POTENTIAL  GTH-PBE-q6
    &END KIND
    &KIND H
      BASIS_SET  DZVP-MOLOPT-GTH
      POTENTIAL  GTH-PBE-q1
    &END KIND
  &END SUBSYS
&END FORCE_EVAL

7. 解析方法

振動スペクトル（VDOS）の計算

速度自己相関関数（VACF）のフーリエ変換から振動状態密度（VDOS）を得られます：

$$g(\omega) = \int_0^\infty \langle \mathbf{v}(t) \cdot \mathbf{v}(0) \rangle \cos(\omega t)\, dt$$
vdos.py — VACF と VDOS の計算
import numpy as np
import matplotlib.pyplot as plt
from ase.io import read

# VASP の vasprun.xml または XYZ トラジェクトリを読み込む
traj = read("vasprun.xml", index=":", format="vasp-xml")
dt = 0.5e-15   # 時間刻み [s]

# 速度を取得（[nstep, natom, 3]）
vel = np.array([atoms.get_velocities() for atoms in traj])

# VACF（全原子平均）
nstep, natom, _ = vel.shape
vacf = np.zeros(nstep)
for lag in range(nstep):
    vacf[lag] = np.mean(np.sum(vel[:nstep-lag] * vel[lag:], axis=-1))

# フーリエ変換 → VDOS
vdos = np.abs(np.fft.rfft(vacf))**2
freq = np.fft.rfftfreq(nstep, d=dt)
freq_cm = freq / (3e10)   # Hz → cm⁻¹

plt.plot(freq_cm, vdos)
plt.xlabel("Wavenumber (cm⁻¹)")
plt.ylabel("VDOS (a.u.)")
plt.xlim(0, 4000)
plt.tight_layout()
plt.savefig("vdos.png", dpi=150)
plt.show()

平均二乗変位（MSD）と拡散係数

$$\text{MSD}(t) = \langle |\mathbf{r}(t) - \mathbf{r}(0)|^2 \rangle，\quad D = \lim_{t\to\infty} \frac{\text{MSD}(t)}{6t}$$
