1. 局所秩序変数（LOP）とは

局所秩序変数（Local Order Parameter, LOP）は， MD シミュレーションの各原子・分子の局所的な幾何学的配置を定量化するスカラー量です． 各粒子に対して，その近傍粒子との相対的な位置・角度関係から1つの数値が計算されます．

LOP の主な用途：

  • 相の識別：液体・結晶・非晶質・気体を自動的に分類
  • 相変化の追跡：融解・凝固・結晶化の過程をリアルタイムで可視化
  • 核生成の検出：臨界核の出現と成長を捉える
  • 構造特性評価：積層欠陥，粒界，転位などの欠陥構造を特定
なぜ LOP が必要か： MD トラジェクトリには膨大な原子座標が記録されていますが， 人間の目で「どこが結晶でどこが液体か」を判断することは難しいです． LOP は各原子に数値ラベルを付けることで，自動的に相を分類・可視化できます．

2. 代表的な LOP

ボンド秩序変数（Steinhardt パラメータ）

最も広く使われる LOP の一つです．球面調和関数を使って近傍粒子との結合ベクトルの角度分布を特徴付けます：

$$q_l(i) = \sqrt{\frac{4\pi}{2l+1}\sum_{m=-l}^{l}\left|\bar{q}_{lm}(i)\right|^2}$$

ここで

$$\bar{q}_{lm}(i) = \frac{1}{N_b(i)}\sum_{j=1}^{N_b(i)} Y_l^m(\hat{\mathbf{r}}_{ij})$$

$Y_l^m$ は球面調和関数，$N_b(i)$ は粒子 $i$ の近傍数，$\hat{\mathbf{r}}_{ij}$ は結合ベクトルの単位ベクトル．

$l$ の値感度のある構造
$q_4$FCC / HCP / 液体の区別
$q_6$FCC / BCC / 液体 / 結晶一般
$q_3$正四面体配位（氷・ダイヤモンド構造）

CHILL+ パラメータ（氷・ハイドレート向け）

氷の多形・クラスレートハイドレート・液体水を区別するために最適化された LOP です． 近傍間の結合のコサイン相関を用いて，氷 Ih, Ic，クラスレートハイドレート，液体水を分類できます．

CNA（Common Neighbor Analysis）

2粒子の共有近傍数・結合数・最長鎖長を指標として FCC / HCP / BCC / ICO を高精度に分類します． OVITO や LAMMPS に組み込まれています．

3. MALIO の概要

MALIO（MAchine Learning-assisted Investigation of local Order parameters）は， 高橋和義（産業技術総合研究所）が開発した， MD トラジェクトリから最適な LOP を自動探索するフレームワークです．

従来は研究者が手動で LOP の種類（$q_l$, CNA 等）とパラメータを選んでいましたが， MALIO では機械学習（分類器）とパラメータ空間の探索を組み合わせて， 指定した構造を最もよく識別できる LOP とそのパラメータを自動的に見つけます．

MALIO のアルゴリズム概要

1
教師データの準備：識別したい相（例：液体・結晶）のスナップショットを用意し，各原子に相ラベルを付与
2
LOP 候補の生成：多数の LOP タイプ（$q_l$, $w_l$, 角度分布など）とパラメータ（$l$, カットオフ距離等）を自動的にスキャン
3
分類性能の評価：各 LOP について分類器（線形 SVM または決定木）で識別精度を評価
4
最適 LOP の出力：識別精度が最高の LOP とそのパラメータを推奨として出力
5
全トラジェクトリへの適用：見つかった LOP を使って全時系列を解析・可視化

4. MALIO の使い方

MALIO は Python パッケージとして提供されています．インストールと基本的な使い方：

Terminal — インストール
# GitHub からインストール
pip install git+https://github.com/KZ-Takahashi/malio.git

# または conda から（準備中）
conda install -c conda-forge malio
malio_example.py — 基本的な使い方
import malio
from ase.io import read

# ── 1. 教師データの準備 ─────────────────────────────────
# 液体と結晶の参照スナップショット
liquid_frames = read("liquid.xyz", index=":")
crystal_frames = read("crystal.xyz", index=":")

# ── 2. MALIO でLOP を探索 ────────────────────────────────
searcher = malio.LOPSearcher(
    cutoff_range=(3.0, 7.0),   # カットオフ探索範囲 [Å]
    l_range=(2, 8),              # 球面調和関数の次数
    classifier="linear_svm",   # 分類器の種類
)

results = searcher.fit(
    reference_sets={
        "liquid": liquid_frames,
        "crystal": crystal_frames,
    }
)

# ── 3. 結果の確認 ─────────────────────────────────────
best_lop = results.best_lop
print(f"最適LOP: {best_lop.name}, l={best_lop.l}, rc={best_lop.cutoff:.2f} Å")
print(f"識別精度: {results.best_score:.4f}")
results.plot_ranking()   # 上位 LOP のランキングをプロット

# ── 4. 全トラジェクトリへの適用 ───────────────────────
traj = read("md.xyz", index=":")
lop_values = best_lop.compute(traj)   # 各原子・各フレームの LOP 値

# フレームごとの相分率を計算
labels = best_lop.classify(traj)      # 0=liquid, 1=crystal
crystal_fraction = labels.mean(axis=1)
print("結晶相分率の時系列:", crystal_fraction)

5. 研究室での応用例

ジブロックコポリマーのミクロ相分離構造の識別（2024）

高野・平塚・高橋（Scientific Reports, 2024）では，MALIO を用いて ジブロックコポリマーのラメラ・ジャイロイド・シリンダーなどのミクロ相分離構造を MD トラジェクトリから自動的に識別する LOP を探索しました． 従来の構造因子や密度場解析より高感度・高速に相を識別できることを示しました．

氷・クラスレートハイドレートの水ネットワーク識別（2023）

高橋・平塚（Crystal Growth & Design, 2023）では， 氷（Ih・Ic）とシクロプロパンクラスレートハイドレートの水分子ネットワークを 識別する最適 LOP を MALIO で探索しました． $q_3$ と独自の角度相関 LOP の組み合わせが最も効果的であることが分かりました．

結晶性高分子のラメラ構造識別（2022）

高野・平塚ら（JCP, 2022）では，ポリエチレンの MD トラジェクトリにおいて 結晶ラメラと非晶質部分を識別する LOP を MALIO で探索． 六方格子構造に感度を持つ $q_6$ の変種が最適であることを示しました．

6. Python での LOP 計算例（OVITO 使用）

OVITO の Python API を使うと，ボンド秩序変数や CNA を簡単に計算できます：

lop_ovito.py — OVITO で $q_6$ を計算・可視化
import ovito
from ovito.io import import_file
from ovito.modifiers import BondAngleAnalysisModifier, PolyhedralTemplateMatchingModifier
from ovito.modifiers import BondOrderParameterModifier
import numpy as np
import matplotlib.pyplot as plt

# トラジェクトリの読み込み
pipeline = import_file("dump.lammpstrj")

# ボンド秩序変数 q6 の計算
pipeline.modifiers.append(
    BondOrderParameterModifier(
        l=6,
        cutoff=3.8,           # カットオフ [Å]
        compute_global=True,
    )
)

# Polyhedral Template Matching（FCC/HCP/BCC の自動識別）
ptm = PolyhedralTemplateMatchingModifier()
pipeline.modifiers.append(ptm)

# 全フレームを処理
q6_mean_list = []
for frame in range(pipeline.source.num_frames):
    data = pipeline.compute(frame)
    q6 = data.particles["q6"]
    q6_mean_list.append(np.mean(q6))

plt.plot(q6_mean_list)
plt.xlabel("Frame")
plt.ylabel("Mean q6")
plt.title("Crystallization tracking via q6")
plt.savefig("q6_tracking.png", dpi=150)
plt.show()
