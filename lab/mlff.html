1. ML-FF とは

機械学習力場（Machine Learning Force Field, ML-FF）は，DFT などの第一原理計算データを 教師データとして機械学習モデルを訓練し，古典 MD と同程度の計算コストで 第一原理精度に近いエネルギー・力を予測する手法です．

ML-FF の位置づけ：古典 MD（速い・精度低）と AIMD（遅い・精度高）の間を埋める手法． 原子数 10³〜10⁵ 規模で DFT 精度を実現できます．

ML-FF が予測する物理量は：

$$E_\text{tot} = \sum_i \varepsilon_i(\{r_{ij}, Z_j\}_{j \in \mathcal{N}(i)})$$

原子 $i$ のエネルギー寄与 $\varepsilon_i$ は，その近傍原子の配置（局所環境）のみに依存すると仮定します（局所性の原理）．

3. MACE の理論

MACE（Multi Atomic Cluster Expansion）は Batatia らが開発した等変グラフニューラルネットワークです． E(3) 等変性（回転・反転・並進対称性）を厳密に満たし，少ない訓練データで高精度を達成します．

各原子の特徴ベクトルはメッセージパッシングで更新されます：

$$\mathbf{h}_i^{(t+1)} = f_\text{update}\!\left(\mathbf{h}_i^{(t)},\; \sum_{j \in \mathcal{N}(i)} f_\text{msg}\!\left(\mathbf{h}_i^{(t)}, \mathbf{h}_j^{(t)}, \mathbf{r}_{ij}\right)\right)$$

等変メッセージには球面調和関数 $Y_l^m(\hat{\mathbf{r}}_{ij})$ が使われ， 回転変換に対して正しく変換するテンソル特徴量として扱われます．

4. MACE-MP-0（汎用力場）

MACE-MP-0は Materials Project の DFT データ（〜150,000 構造）で訓練された 汎用 ML-FF です．元素周期表の大部分の元素をカバーし，特別な訓練なしに多くの系に適用できます．

インストール

Terminal
pip install mace-torch

基本的な使い方（ASE 経由）

mace_mp_example.py
from mace.calculators import mace_mp
from ase.build import bulk
from ase.optimize import BFGS
from ase import units
from ase.md.nvtberendsen import NVTBerendsen
from ase.io.trajectory import Trajectory

# MACE-MP-0 カルキュレーターの設定
# dispersion=True で DFT-D3 補正を追加
calc = mace_mp(model="medium", dispersion=True, device="cpu")

# 構造の作成（例：FCC アルミニウム）
atoms = bulk("Al", "fcc", a=4.05) * (3, 3, 3)
atoms.calc = calc

# 構造最適化
opt = BFGS(atoms, logfile="opt.log")
opt.run(fmax=0.01)   # eV/Å
print(f"Energy: {atoms.get_potential_energy():.4f} eV")

# NVT MD（Berendsen サーモスタット）
atoms.set_momenta(
    [0] * (len(atoms) * 3)  # まずゼロ初期化
)
from ase.md import velocitydistribution as vd
vd.MaxwellBoltzmannDistribution(atoms, temperature_K=300)

dyn = NVTBerendsen(atoms, timestep=1.0*units.fs,
                   temperature_K=300, taut=100*units.fs)
traj = Trajectory("md.traj", "w", atoms)
dyn.attach(traj.write, interval=10)
dyn.run(1000)   # 1000 ステップ = 1 ps
print("MD done")
モデルサイズ： "small"（速い），"medium"（推奨），"large"（高精度・遅い）の3種類． GPU がある場合は device="cuda" で大幅に高速化できます．

5. MACE のファインチューニング

汎用モデル MACE-MP-0 を出発点として，特定の系の DFT データで追加訓練（ファインチューニング）することで 高精度な専用力場を作成できます．

訓練データの作成フロー

1
AIMD 計算（VASP/CP2K）で対象系のトラジェクトリを生成（数百〜数千構造）
2
データセット作成：各構造のエネルギー・力・応力を DFT で計算し，XYZ 形式に変換
3
訓練・検証・テスト分割（例：8:1:1）
4
ファインチューニング実行
Terminal — MACE のファインチューニング
mace_run_train \
  --name="hydrate_mace" \
  --foundation_model="medium" \
  --train_file="train.xyz" \
  --valid_file="valid.xyz" \
  --test_file="test.xyz" \
  --energy_weight=1.0 \
  --forces_weight=10.0 \
  --stress_weight=1.0 \
  --max_num_epochs=200 \
  --batch_size=4 \
  --device=cuda \
  --seed=42

6. LAMMPS との連携

訓練済みの MACE モデルを LAMMPS から呼び出して大規模 MD を行えます． pair_style mace が LAMMPS に組み込まれています．

LAMMPS input — MACE 力場の使用例
pair_style   mace no_domain_decomposition
pair_coeff   * * hydrate_mace.model O H C

7. Active Learning（能動学習）

ML-FF の精度は訓練データの網羅性に依存します． Active Learningでは，ML-FF の不確実性が高い構造（未知の化学空間）を 自動的に DFT で再計算し，訓練データに追加する反復プロセスで力場を改善します．

1
初期 ML-FF で MD を実行
2
不確実性（アンサンブル分散）が閾値を超えた構造を抽出
3
抽出構造の DFT 計算を実行
4
データを追加して ML-FF を再訓練
5
収束するまで繰り返す
