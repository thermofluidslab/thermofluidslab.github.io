1. VASP とは

VASP（Vienna Ab initio Simulation Package）は，プレーンウェーブ基底と PAW（Projector Augmented Wave）法を用いた DFT コードです． 固体・表面・分子のエネルギー最小化・電子状態・AIMD に広く使われています． 商用ライセンスのため，大学・機関単位でライセンスを取得する必要があります．

ライセンス：VASP は商用ソフトです．ライセンスなしでの使用は禁止されています． 指導教員に使用許可を確認してください．

INCAR — 計算パラメータ

INCAR — 構造最適化
SYSTEM = Ice Ih optimization

# 基本設定
ISTART = 0
ICHARG = 2
ENCUT  = 520       # プレーンウェーブカットオフ [eV]
EDIFF  = 1E-6      # SCF 収束条件
PREC   = Accurate

# 汎関数 + vdW 補正
GGA    = PE        # PBE
IVDW   = 11        # DFT-D3

# 構造最適化
IBRION = 2         # 共役勾配法
NSW    = 200        # 最大ステップ数
ISIF   = 3         # 全体積・形状・内部座標を最適化
EDIFFG = -0.01     # 力の収束条件 [eV/Å]

# スメアリング
ISMEAR = 0         # ガウス型（絶縁体・分子向け）
SIGMA  = 0.05

# 出力
LWAVE  = .FALSE.
LCHARG = .FALSE.

KPOINTS — k 点サンプリング

KPOINTS — Monkhorst-Pack グリッド
Automatic mesh
0              ! 自動生成
Monkhorst-Pack
4 4 4          ! k点グリッド
0 0 0          ! シフト
分子・大規模セルの場合： Γ 点のみで十分な場合は 1 1 1（またはKSPACING=1000をINCAR に記載）を使用します． ハイドレートの AIMD では通常 Γ 点のみです．

POTCAR — PAW ポテンシャル

Terminal — POTCAR の作成
# POSCAR の元素順に POTCAR を結合
cat $VASP_PP_PATH/PAW_PBE/O/POTCAR \
    $VASP_PP_PATH/PAW_PBE/H/POTCAR > POTCAR

3. 構造最適化

Terminal — 実行とモニタリング
# 実行（並列）
mpirun -np 16 vasp_std > vasp.out 2>&1 &

# 収束状況のモニタリング
tail -f OSZICAR

# 力の収束確認
grep "reached required accuracy" OUTCAR

# 最適化後の構造確認（ASE）
python -c "from ase.io import read; a=read('CONTCAR'); print(a)"

4. 電子状態計算（DOS・バンド）

状態密度（DOS）

INCAR — DOS 計算
ISTART = 1         # 構造最適化の WAVECAR を継続使用
ICHARG = 11        # 電荷密度を読んで固定（CHGCAR から）
NSW    = 0
IBRION = -1
ISMEAR = -5        # テトラへドロン法（正確な DOS に推奨）
LORBIT = 11        # 部分 DOS を出力（DOSCAR）
NEDOS  = 2000      # DOS の点数

pyiron / py4vasp による解析（Python）

read_dos.py
from vaspkit_tools import Doscar   # または pyiron
import numpy as np
import matplotlib.pyplot as plt

# ASE でも DOSCAR を読める
from ase.calculators.vasp import VaspChargeDensity
from ase.dft.dos import DOS

calc = VaspChargeDensity("CHGCAR")
dos = DOS(calc, width=0.2)
e, d = dos.get_dos()
plt.plot(e, d)
plt.xlabel("Energy (eV)")
plt.ylabel("DOS (states/eV)")
plt.savefig("dos.png", dpi=150)

6. 出力ファイルの解析

ファイル内容読み込み方
OUTCAR全計算情報（力，ストレス，振動数等）grep または ASE
CONTCAR最終構造（最適化後の POSCAR 形式）ase.io.read("CONTCAR")
vasprun.xml全情報の XML（トラジェクトリ等）ase.io.read("vasprun.xml", ":")
EIGENVALバンド固有値pyiron, vaspkit
DOSCAR状態密度ASE, pyiron
XDATCARMD トラジェクトリ（AIMD 用）ase.io.read("XDATCAR", ":")
