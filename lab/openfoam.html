1. OpenFOAM とは

OpenFOAM（Open Field Operation And Manipulation）は， 有限体積法（FVM）ベースのオープンソース CFD フレームワークです． C++ で記述されており，ソルバーのカスタマイズが可能です． 定常・非定常，圧縮性・非圧縮性，乱流・層流，熱伝達，多相流など幅広い問題に対応しています．

ディストリビューション開発元特徴
OpenFOAM.orgOpenFOAM Foundationコア開発版
OpenFOAM.com (ESI)ESI Group機能が豊富，企業サポートあり
foam-extendコミュニティ実験的機能多数

sudo add-apt-repository http://dl.openfoam.org/ubuntu
sudo apt update
sudo apt install openfoam12

# 環境変数の設定（毎回 source するか ~/.bashrc に追記）
source /opt/openfoam12/etc/bashrc

# 動作確認

3. ケース構造

myCase/ ├── 0/ ← 初期条件（時刻 0 の場の値） │ ├── U 速度場 │ ├── p 圧力場 │ ├── k 乱流エネルギー │ └── omega 乱流周波数（k-ω SST 使用時） ├── constant/ ← 物性・メッシュ情報 │ ├── transportProperties 粘度など │ ├── turbulenceProperties 乱流モデルの選択 │ └── polyMesh/ メッシュデータ └── system/ ← 計算設定 ├── controlDict 実行制御（時間刻み・出力設定） ├── fvSchemes 離散化スキーム └── fvSolution 線形ソルバー設定

4. 定常計算（simpleFoam）

simpleFoam は非圧縮性定常乱流ソルバーです（SIMPLE アルゴリズム）． 主に外部空気力学（車，スポーツ球体）や室内換気の定常解析に使います．

controlDict（定常）

system/controlDict
application     simpleFoam;

startFrom       startTime;
startTime       0;
stopAt          endTime;
endTime         500;           // 反復回数

deltaT          1;             // 定常計算では 1
writeControl    timeStep;
writeInterval   50;
purgeWrite      3;             // 最新3つのみ保存

runTimeModifiable yes;

functions
{
    forceCoeffs
    {
        type        forceCoeffs;
        libs        ( forces );
        patches     ( ball );       // 力を求める面
        rho         rhoInf;
        rhoInf      1.225;          // 空気密度 [kg/m³]
        liftDir     (0 1 0);
        dragDir     (1 0 0);
        CofR        (0 0 0);
        pitchAxis   (0 0 1);
        magUInf     20;             // 代表速度 [m/s]
        lRef        0.22;           // 代表長さ（ボール直径）
        Aref        0.038;          // 代表面積 [m²]
    }
}

0/U（速度場の初期・境界条件）

0/U
dimensions      [0 1 -1 0 0 0 0];   // m/s

internalField   uniform (20 0 0);   // 初期値

boundaryField
{
    inlet
    {
        type            fixedValue;
        value           uniform (20 0 0);
    }
    outlet
    {
        type            zeroGradient;
    }
    ball
    {
        type            noSlip;     // 壁面すべり無し
    }
    sides
    {
        type            slip;       // 外部境界はすべり
    }
}

実行

Terminal
blockMesh           # メッシュ生成
checkMesh           # メッシュ品質確認
simpleFoam          # 計算実行
simpleFoam | tee log.simpleFoam  # ログを保存しながら実行

# 並列計算（4コア）
decomposePar
mpirun -np 4 simpleFoam -parallel
reconstructPar

5. 非定常計算（pimpleFoam）

pimpleFoam は非圧縮性非定常乱流ソルバー（PIMPLE = PISO + SIMPLE アルゴリズム）です． 後流の渦放出，球体の非定常揚力など時間変化する現象に使います．

system/controlDict（非定常）
application     pimpleFoam;

startTime       0;
endTime         2.0;            // [s]
deltaT          1e-4;           // 時間刻み [s]
adjustTimeStep  yes;
maxCo           0.8;            // Courant 数の上限

writeControl    adjustableRunTime;
writeInterval   0.01;           // 0.01 秒ごとに出力
Courant 数（CFL 条件）： $Co = U \Delta t / \Delta x \leq 1$ が安定条件の目安です． OpenFOAM では adjustTimeStep yes で自動調整できます．

6. メッシュ生成

blockMesh（構造格子）

system/blockMeshDict — 直方体ボックスの例
scale   1;

vertices
(
    (-1.0 -0.5 -0.5)  // 0
    ( 2.0 -0.5 -0.5)  // 1
    ( 2.0  0.5 -0.5)  // 2
    (-1.0  0.5 -0.5)  // 3
    (-1.0 -0.5  0.5)  // 4
    ( 2.0 -0.5  0.5)  // 5
    ( 2.0  0.5  0.5)  // 6
    (-1.0  0.5  0.5)  // 7
);

blocks
(
    hex (0 1 2 3 4 5 6 7)
    (60 20 20)                  // セル数 (x y z)
    simpleGrading (1 1 1)
);

boundary
(
    inlet  { type patch; faces ((0 4 7 3)); }
    outlet { type patch; faces ((1 2 6 5)); }
    sides  { type patch; faces ((0 1 5 4)(3 7 6 2)(0 3 2 1)(4 5 6 7)); }
);

snappyHexMesh（STL 形状のメッシュ化）

概要
# 1. STL ファイルを constant/triSurface/ に配置
# 2. system/snappyHexMeshDict を設定
# 3. 実行
blockMesh           # 背景メッシュ
surfaceFeatureExtract  # 稜線抽出
snappyHexMesh -overwrite  # メッシュ生成

7. 後処理（ParaView）

Terminal
# ParaView 用の .foam ファイルを作成
touch case.foam

# ParaView を起動して case.foam を開く
paraview case.foam &

ParaView での主な操作：

  • StreamTracer：流線の可視化
  • Glyph：速度ベクトルの表示
  • Contour：等値面（圧力，速度等）
  • Slice：断面表示
  • PlotOverLine：線上のデータをグラフ化
