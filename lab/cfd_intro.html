CFD流体力学乱流

1. CFD とは

数値流体力学（Computational Fluid Dynamics, CFD）は， 流体の支配方程式（連続の式，Navier–Stokes 方程式，エネルギー方程式）を 数値的に離散化・求解することで，流れ場を計算する手法です．

当研究室の CFD 応用例：車椅子マラソンの熱流体解析，トンネル換気，スポーツ球体の空気力学，水泳の流れ解析など．

手法特徴用途
RANS（定常・非定常）乱流を時間平均，計算コスト小工学的設計，外部流れ
LES（Large Eddy Simulation）大渦を直接解像，中程度コスト剥離流れ，後流，音響
DNS（直接数値シミュレーション）全スケール解像，計算コスト最大乱流研究，基礎研究

2. 支配方程式

連続の式（質量保存）

$$\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{u}) = 0$$

非圧縮性流れ（$\rho = \text{const.}$）では：

$$\nabla \cdot \mathbf{u} = 0$$

Navier–Stokes 方程式（運動量保存）

$$\rho \frac{D\mathbf{u}}{Dt} = \rho\left(\frac{\partial \mathbf{u}}{\partial t} + (\mathbf{u}\cdot\nabla)\mathbf{u}\right) = -\nabla p + \mu \nabla^2 \mathbf{u} + \rho\mathbf{g}$$

左辺：慣性項（移流項 + 時間微分），右辺：圧力勾配，粘性項，体積力

エネルギー方程式

$$\rho c_p \frac{DT}{Dt} = \nabla \cdot (k \nabla T) + \dot{q}$$

無次元パラメータ

無次元数定義物理的意味
Reynolds 数 $Re$$\rho U L / \mu$慣性力 / 粘性力（層流・乱流の指標）
Mach 数 $Ma$$U / c$流速 / 音速（圧縮性の指標）
Nusselt 数 $Nu$$hL / k$対流 / 伝導熱伝達の比
Prandtl 数 $Pr$$\mu c_p / k$運動量 / 熱拡散の比
Strouhal 数 $St$$fL / U$振動現象の無次元周波数

3. 離散化手法

手法略称特徴使用例
有限体積法FVM保存則を厳密に満足，任意形状対応OpenFOAM, FLUENT
有限差分法FDM実装が単純，構造格子向けDNS コード
有限要素法FEM複雑形状・構造解析と親和性高COMSOL, Code_Aster
スペクトル法—高精度，周期境界問題向けDNS, Dedalus

有限体積法の基本

計算領域を制御体積（CV）に分割し，各 CV について積分形支配方程式を離散化：

$$\frac{\partial}{\partial t}\int_V \phi\, dV + \oint_S \phi \mathbf{u}\cdot d\mathbf{S} = \oint_S \Gamma \nabla\phi \cdot d\mathbf{S} + \int_V S_\phi\, dV$$

面積分を各面の値で近似することで代数方程式系に変換します．

4. 乱流モデル

RANS モデル

Reynolds 平均 NS 方程式（RANS）には Reynolds 応力テンソル $\overline{u_i' u_j'}$ が出現し， これを「閉じる」ためにモデルが必要です（乱流クロージャ問題）．

モデル特徴用途
$k$-$\varepsilon$ 標準壁面境界層に注意が必要一般的な内部流れ
$k$-$\omega$ SST壁面近傍と自由流れの両方に適合外部空気力学（推奨）
Spalart-Allmaras1 方程式，航空機設計向け翼周り流れ
RSM（Reynolds Stress）高精度だが収束が難しい旋回流れ，二次流れ

壁面処理

乱流の壁面近傍では粘性底層（$y^+ < 5$）が存在します． 壁関数を使う場合は $30 < y^+ < 300$ が適切， 低 Reynolds 数モデルを使う場合は $y^+ \approx 1$ が必要です．

スポーツ球体のシミュレーションでは： 剥離点の位置が重要なため，$k$-$\omega$ SST + 低 $y^+$ メッシュが推奨されます． 臨界 Reynolds 数付近では LES も有効です．

5. メッシュ生成

ツール種別特徴
blockMesh（OpenFOAM）構造格子シンプル形状に適した入力ベースのメッシャー
snappyHexMesh（OpenFOAM）非構造STL 形状を自動的にメッシュ化，並列対応
GMSH非構造（FEM/FVM）オープンソース，Python API あり
ANSYS Meshing非構造商用，GUI 操作，高品質
Pointwise / Fidelity Pointwise構造/非構造航空宇宙向け高品質メッシュ

メッシュ品質の指標

  • 直交性（Orthogonality）：隣接セル間のベクトルと面法線の角度（90° が理想）
  • スキュー性（Skewness）：セルの歪み（0 に近いほど良い）
  • アスペクト比（Aspect Ratio）：セルの辺長比（境界層では大きくなる）
  • $y^+$：壁面近傍の無次元距離（乱流モデルに応じた値が必要）

6. 境界条件

境界条件内容OpenFOAM での設定
inlet（流入）速度・乱流強度を指定fixedValue for U
outlet（流出）圧力を指定（速度はゼロ勾配）zeroGradient for U, fixedValue for p
wall（壁面）すべり無し条件（$\mathbf{u}=0$）noSlip
symmetry（対称）対称平面（法線方向速度=0）symmetry
cyclic（周期）周期境界（入口と出口をペア）cyclic
